# LumiNails Agenda

Aplicação Android nativa para gestão de clientes, serviços e marcações da manicure Patrícia, com foco em rapidez e usabilidade no dia a dia.

## Stack
- Kotlin
- Jetpack Compose
- MVVM
- Room (offline-first)
- GitHub Actions (build automático de APK)

## Como obter o APK no GitHub Actions
1. Abrir o separador **Actions** no repositório.
2. Entrar na execução do workflow **Build Android APK**.
3. Em **Artifacts**, descarregar **luminails-debug-apk**.
